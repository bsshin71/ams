# ams
Graphical monitoring system for ALTIBASE

Build with using InfluxDB, Telegraf, Grafana





# 1. Open Source 기반으로 제작된  Altibase  모니터링 솔류션을 상용 솔류션 대비 저렴한 가격으로  제공합니다.  

별도의  상용 s/w를 사용하지 않아   구축 컨설팅 비용외 추가 비용  부담 없음.
